四、安装与使用
 
1. 下载安装
 
- 官网直达：https://xmail.34567.xin
- 选择对应操作系统版本（当前仅支持 Windows 10/11），下载后双击安装包，跟随引导完成安装（无需复杂配置）
 
2. 邮箱绑定教程
 
1. 打开 XMail Pro，点击「添加账号」，选择对应邮箱服务商（如 Gmail、QQ邮箱）
2. 输入邮箱账号，密码栏需填写「客户端授权码」（而非邮箱主密码，授权码获取教程见下文）
3. 验证通过后，账号自动同步邮件，即可开始使用
 
3. 授权码获取说明
 
绑定邮箱需使用「客户端授权码」（第三方邮件工具通用要求），获取步骤如下：
 
1. 登录对应邮箱的网页版（如 QQ邮箱 mail.qq.com、Gmail mail.google.com）
2. 进入「设置」→「账户安全」/「客户端设置」板块
3. 开启「客户端授权码」/「应用专用密码」功能（部分邮箱需绑定手机号验证）
4. 验证身份后生成授权码，复制并粘贴到 XMail Pro 密码栏即可
 
- 注：不同邮箱操作路径略有差异，遇问题可参考群公告或联系技术支持
