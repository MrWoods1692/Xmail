下载 → 双击安装 → 完成即可使用，无需额外配置。


### 2. 添加邮箱（以 QQ 为例）
- 启动 XMail Pro → 「添加账号」→ 选择服务商  
- 输入邮箱地址  
- 密码栏粘贴 “客户端授权码”（非邮箱登录密码）  
- 验证通过 → 邮件自动同步

> 授权码获取：网页登录邮箱 → 设置 → 账户安全 → 生成“客户端授权码”/“应用专用密码”。
